My Projects

TrailBlazerUpKeeper

TrailBlazerUpKeeper is a project I made for the Utah Tech University Building Maintenance crew

The purpose of the project was to help keep track of the many paints on campus.

Previously Building Maintenance employees had to either remember themselves what a certain room was painted or they just had to guess. I grew frustrated with having to guess and always being wrong. So I built TrailBlazerUpKeeper.

If you wanted to check out how I built this, you can click Here.

I also have this project deployed, i used render.com which is free so it takes a little while for the backend to show up. You can see it here at trailblazerupkeeper.com

Weather Station

This was a project for CS4990 Embedded Systems, where i made a weather station that could read the temperature and humidity in an area and then transmit that data.

I made a web application that displays the data on the web and updates in real time.

This project uses Lora which allowed for low-power, long-range data transmission to send my data.
